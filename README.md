# Monocular Visual Odometry

Contains simple Python and C++ scripts to perform Monocular Visual odometry using OpenCV on the KITTI odometry dataset. 

## Usage

C++

```
mkdir build
cmake ..
make
./vo
```
Python

```
python vo.py
```
